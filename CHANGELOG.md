# 变更日志

## [0.0.4] - 2026-06-25
- 上一个版本的 alt+oem_3 命令有 bug，当点击扩展，然后查看扩展详细页时，该命令失效。同时我发现没必要实现再次点击可以关闭侧边栏的显示，麻烦，而且和 ctrl+b 快捷键重复了
- 添加变更日志，让每次发布 release 时可以自行生成变更内容

## [0.0.3] - 2026-06-25
- 修改了 alt+oem_3 命令，让其可以聚焦到具体文件，而不是只聚焦到目录树，同时设置再次点击可以关闭侧边栏的显示
- 添加了 action，每次提交新版本时自动发布到vscode扩展商城，并且在 release 中附上扩展安装包
